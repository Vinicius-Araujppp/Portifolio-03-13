### 🍽️ ClickServe — PI 4° Semestre
> **Sistema completo de gestão para restaurantes** — Projeto Integrador IV, Grupo 04, DSM/FATEC Franca 2025.2.

Plataforma com 3 camadas integradas em tempo real via WebSocket:
- **Web** (React): painel gerencial — usuários, mesas, cardápio, relatórios financeiros.
- **Mobile** (React Native + Expo): app operacional para garçons e cozinha — pedidos, chamados, notificações e pagamentos em tempo real.
- **Backend** (Node.js + Express + MongoDB + Socket.IO): API RESTful + WebSocket com autenticação JWT.

**Tecnologias:** React • Vite • Tailwind CSS • React Native • Expo • TypeScript • Node.js • Express • MongoDB • Mongoose • Socket.IO • JWT • Bcrypt  
🔗 [github.com/Vinicius-Araujppp/ClickServe22](https://github.com/Vinicius-Araujppp/ClickServe22)
